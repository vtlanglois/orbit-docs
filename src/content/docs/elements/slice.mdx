`<o-slice>` is a standard and vanilla web-component for rendering a radial slices, arcs or pies. By default there are up to 60 slices elements per orbit. 

## Look and feel


### Changing slice colors
To change o-slice color use CSS custom var `--color` and `--hover-color` in a `o-slice` style.

export const slice = `
<div class="bigbang">
  <div class="gravity-spot">
    <div class="orbit-5">
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
      <o-slice class="cyan"></o-slice>
      <o-slice class="purple"></o-slice>
    </div>
  </div>
</div>
`;
export const sliceCSS = `
o-slice.cyan {

`;

### Changing endings

`o-slice` has one special attribute: `shape` to define different endings. Currently, you can choose between `circle`, `arrow`, `slash`, `backslash` and `zigzag` shapes. Default `none`. 

:::note
Shapes don't work on Safari browsers.
:::

## Adjust o-slice size

Each `o-slice` has one orbit space size, but it can be finely adjusted using the CSS class utility `.shrink-10` to `.shrink-90`, allowing the vectors to shrink by a specified percentage. 


export const shrink = `
<div class="bigbang">
  <div class="gravity-spot">
    <div class="orbit-5">
      <o-slice style="--o-slice-color: var(--o-red)"></o-slice>
      <o-slice class="shrink-90" style="--o-slice-color: var(--o-orange)"></o-slice>
      <o-slice class="shrink-80" style="--o-slice-color: var(--o-yellow)"></o-slice>
      <o-slice class="shrink-70" style="--o-slice-color: var(--o-color)"></o-slice>
      <o-slice class="shrink-60" style="--o-slice-color: var(--o-green)"></o-slice>
      <o-slice class="shrink-50" style="--o-slice-color: var(--o-cyan)"></o-slice>
      <o-slice class="shrink-40" style="--o-slice-color: var(--o-blue)"></o-slice>
      <o-slice class="shrink-30" style="--o-slice-color: var(--o-indigo)"></o-slice>
      <o-slice class="shrink-20" style="--o-slice-color: var(--o-purple)"></o-slice>
      <o-slice class="shrink-10" style="--o-slice-color: var(--o-pink)"></o-slice>
      
    </div>
  </div>
</div>
`;
export const shrinkCSS = `

.orbit-5 {
border: 1px dashed var(--o-cyan);
}
o-slice {
--o-gap: 1; /* This will be added as default soon*/
}

`;

On opposite way, the CSS class utility `.grow-0x` to `.grow-12x`, allowing vector to expand by a multiple of `.orbit` size. It is possible to use decimal fractions. For example,  `.grow-0.1x`,  `.grow-10.7x`, etc.

:::note
`shrink-*` and `.grow-*x` can't be used at same time.
:::

### Relative vector orbital position

By default, `.vector` is positioned in the middle of orbit path. To change that, there some CSS utilities:

**`.inner-orbit`:** To place an`.vector` just below its orbit.

**`.outer-orbit`:** To place `.vector` just above its orbit.

**`.quarter-inner-orbit`:** To place `.vector` a 25% into its orbit.

**`.quarter-outer-orbit`:** To place `.vector` a 25% outer its orbit.

export const inner = `

  <div class="bigbang">
    <div class="gravity-spot">
      <div class="orbit-3">
        <o-slice class="outer-orbit "></o-slice>
        <o-slice class="quarter-outer-orbit"></o-slice>
        <o-slice class="quarter-inner-orbit"></o-slice>
        <o-slice class="inner-orbit"></o-slice>
      </div>
     
  </div>

`;
export const innerCSS = `
.wrapper {
border: 1px dotted var(--o-purple);
height: 250px;
width: 250px;
float: left;
overflow: hidden;
margin: 5px;
}

o-slice {
--o-slice-color: var(--o-pink);
--o-gap: 1; /* This will be added as default soon*/
}



`;
